
# System Actors

Definitions of users and systems that interact with Docgraph.

<a id="ACT-USER"></a>

## User

A human user who interacts with the Docgraph CLI or VS Code extension.

<a id="ACT-CI"></a>

## CI System

An automated Continuous Integration system that runs Docgraph checks.

<a id="ACT-DEV"></a>

## Developer

A software developer who writes code and specifications, and runs Docgraph locally.

<a id="ACT-AGENT"></a>

## AI Agent

An AI agent that consumes the document graph to build knowledge and assist users.
