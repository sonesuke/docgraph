# Human Users

<a id="ACT_USER"></a>

## User

A human user who interacts with the Docgraph CLI or VS Code extension.

### Responsibilities

- Read documentation.
- Browse the graph.

<a id="ACT_DEV"></a>

## Developer

A software developer who writes code and specifications, and runs Docgraph locally.

### Responsibilities

- Write specifications.
- Maintain relationships.
