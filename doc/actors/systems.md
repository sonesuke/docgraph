# System Actors

## CI System

An automated Continuous Integration system that runs Docgraph checks.

### Responsibilities

- Run automated tests.
- Verify relationship rules.

### Participates in

- [UC_CLI_ANALYSIS (CLI Traceability Analysis)](../usecases/cli-analysis.md#UC_CLI_ANALYSIS)

## AI Agent

An AI agent that consumes the document graph to build knowledge and assist users.

### Responsibilities

- Assist in documentation writing.
- Verify rule compliance.

### Participates in

- [UC_AI_ASSISTANCE (AI-Assisted Documentation)](../usecases/ai-assistance.md#UC_AI_ASSISTANCE)
